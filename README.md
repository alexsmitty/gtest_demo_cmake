# gtest_demo_cmake
Example project to show basic gtest config with cmake

# Environment requirements
* cmake
* git
* c++ compiler
* console

# Build With CMake
<pre>
git clone https://github.com/alexsmitty/gtest_demo_cmake.git
cd .\gtest_demo_cmake\
git submodule update --init
mkdir build
cd .\build\
cmake ..
cmake --build .
</pre>
